**Задание**
Создать тест для проверки созданного ранее метода

**Краткое описание процесса выполнения**
1. Создать метод, в нем создать объект сервиса.
2. Создать объект запроса
3. Получить объект ответа через метод сервиса
4. Сравнить поля запроса и ответа

**Что применил/использовал, чему научился в процессе выполнения задания?**
JUnit, Google, вспомнил как пишутся тесты

**Сколько потратил времени на выполнение задачи?**
20-25 минут.

**Как убедился в том, что задание выполнено правильно?**
Тесты пройдены успешно, выполняются не только через их отдельный запуск в IDEA, но и через команду в терминале test.
