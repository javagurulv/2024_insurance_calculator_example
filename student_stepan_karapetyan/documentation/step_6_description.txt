**Задание**
Ознакомиться и попрактиковать возможности команды clean.

**Краткое описание процесса выполнения**
1. Выполнить таску clean для всего проекта через IDEA.
2. Выполнить таску clean для своего модуля через IDEA.
3. Выполнить таску clean для всего проекта через консоль.
4. Выполнить таску clean для своего модуля через консоль.


**Что применил/использовал, чему научился в процессе выполнения задания?**
IDEA и терминал.

**Сколько потратил времени на выполнение задачи?**
3 минуты.


**Как убедился в том, что задание выполнено правильно?**
Сообщения об успешных выполнениях в окне Build, а также убранные скомпилированные папки build(артефакты).
