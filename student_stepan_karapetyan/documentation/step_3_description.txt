**Задание**
Забилдить проект и свой модуль через консоль

**Краткое описание процесса выполнения**
1. Проверка системной переменной JAVA_HOME (Панель управления -> Система -> изменение системных переменных среды -> переменные среды).
2. Если такой переменной нет, то создаем с названием JAVA_HOME, путь устанавливаем к папке, где хранится на пк актуальная JDK).
3. Обязательно перезапускаем IDEA
4. Запуск команды build для всего проекта через консоль
5. Запуск команды build для конкретного модуля через консоль (./gradlew :student_stepan_karapetyan:build)
