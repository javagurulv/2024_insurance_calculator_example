**Задание**
Познакомиться со структурой и файлами после выполнения сборки в Gradle

**Краткое описание процесса выполнения**
Выполнить сборку проекта в Gradle и после выполнения открыть папку build.

**Что применил/использовал, чему научился в процессе выполнения задания?**
Увидел какие создаются артефакты после выполнения сборк:
1. В classes создаются скомпилированные классы. Их структура повторяет структуру проекта
2. В libs лежит собранный jar файл
3. В reports содержится web страница с отчетом прохождения тестов
4. В resources находятся все конфиги и артефакты, взятые из аналогичных пакетов проекта
5. В test-results собрана информация о результатах прохождения тестов в бинарных файлах
6. В tmp лежит временные файлы необходимые для сборки

**Сколько потратил времени на выполнение задачи?**
10 минут.
